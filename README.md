# Hanma Baki VS Kengan Ashura

A fan-made anime crossover website built using HTML, CSS and JavaScript.

## Live Demo
https://kumarsuryavanshiprince-creator.github.io/Hanma-Baki-VS-Kengan-Ashura/

## Features
- Animated loading screen
- Story section
- Character cards with toggle details
- Fight summaries
- Responsive dark theme design

## How to Run Locally
1. Download or clone the repository
2. Open the folder
3. Open `index.html` in any modern web browser

## Deployment
This project is deployed using **GitHub Pages**.
